# How to Create a Simple Website using HTML and CSS

Um projeto simples de **website básico** usando HTML e CSS.

## ✨ Sobre

Este projeto mostra como criar um **site simples e responsivo**, ideal para iniciantes em desenvolvimento web.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Estrutura básica de um site
- Layout organizado
- Design simples e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
